# fflags_comply
O fflags_comply é uma ferramenta inteligente que otimiza o Roblox analisando o hardware do seu PC. Ele identifica sua CPU, GPU e monitor para criar um arquivo ClientAppSettings.json totalmente personalizado. Chega de travar com configurações genéricas: o projeto garante mais FPS, estabilidade e o melhor desempenho que sua máquina pode entregar!
